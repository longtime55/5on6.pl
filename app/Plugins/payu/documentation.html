DOCUMENTATION


PayU (v1.6) plugin for LaraClassified and JobClass.

Installation

To install this plugin:
  • Copy the plugin's (uncompressed) folder into the script's plugins folder like this: /app/Plugins/payu/
  • Go to your Admin panel → Plugins (and click on "Install") to install it.

Compatibility

  • v1.6+ => LaraClassified v6.6+ and JobClass v5.4+
  • v1.5 => LaraClassified (v5.7 to v6.5) and JobClass (v4.5 to v5.3)
  • v1.4 => LaraClassified (v5.2 to v5.6) and JobClass (v4.0 to v4.4)
  • v1.3 => LaraClassified v5.1 and JobClass v3.9
  • v1.2 => LaraClassified (v4.9 to v5.0) and JobClass (v3.7 to v3.8)
  • v1.0 to v1.1 => LaraClassified (v3.1 to v4.8) and JobClass (v2.0 to v3.6)

Update

Replace all the plugin's files on your server.

Configuration

Add the settings below in the script's /.env file on your server: 
PAYU_MODE=
PAYU_POS_ID=
PAYU_SECOND_KEY=
PAYU_OAUTH_CLIENT_SECRET=
Example of use: 
PAYU_MODE=your-payu-mode
PAYU_POS_ID=your-payu-point-of-sale-id
PAYU_SECOND_KEY=your-payu-second-key
PAYU_OAUTH_CLIENT_SECRET=your-payu-oauth-client-secret

Translation

To translate this plugin into your language:
  • Go to the languages folder at /app/Plugins/payu/resources/lang/
  • Duplicate the folder /app/Plugins/payu/resources/lang/en/ to /app/Plugins/payu/resources/lang/<abbr>/
    NOTE: Don't forget replace <abbr> by your language code (e.g. "de" for German language).
  • Open the file /app/Plugins/payu/resources/lang/<abbr>/messages.php and translate it.
    NOTE: To translate the file, you must change the values and not the keys of PHP array.

Uninstallation

To uninstall this plugin, go to your Admin panel → Plugins (and click on "Uninstall") to uninstall it.

Removing

To remove this plugin:
  • Uninstall it
  • Go in the folder /app/Plugins/ and remove the folder payu/.
